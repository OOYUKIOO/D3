Pandas

What is Pandas?

Pandas is a Python module. This library is written for data analysis and management with high performance and high efficiency.

It has a structure similar to spreadsheet, with columns and/or rows. A 1-Dimensional data structure is called series, and a 2-dimensional structure is called dataframe.

Advantages
  • Easy data tracking: All columns and rows are automatically labeled with header and index respectively. Users can make reference to the columns or rows using their corresponding names instead of number.
  • Can contain different data types: Within the same data structure, there can be different types of data, such as integer, float, string, etc.
  • Easy data management: Pandas is equipped with very convenient tools to help you manipulate your data as easily and fast as possible. Some of Pandas' strengths include boolean indexing, pivot table, multi-indexing, crosstab, merge dataframes, sorting dataframes, and graphing. Some of these techniques will be shown later.
